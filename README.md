### to run this (on mac)

- install (love2D's)[https://www.love2d.org/] game engine
- execute the love binary ```sh /Applications/love.app/Contents/MacOS/love ~/path/to/mygame ```
- or set up an alias

```sh
nano ~/.zshrc
```

```sh
alias love="/Applications/love.app/Contents/MacOS/love"
```

```sh
love ~/path/to/mygame
```
